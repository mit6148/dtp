# Team: Jan

## Project: dtp (down to pset)

#### Features implemented

* Log in (by OpenID/MIT Kerberos)

* Create events

* Search events by course, assignment, location, and time

* Add events to schedule

* View schedule

* Edit events

* View number of people attending an event

* Email owner
